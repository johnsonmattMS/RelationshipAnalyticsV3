Utilities

One-off maintenance actions for this record. These run immediately and don’t require a Save — use them when you need to reset or refresh demo data without filling in every tab again.

Roll close date forward

Sets estimatedclosedate on this opportunity to today + N days so the record looks current in demos. Only available on opportunity records.
 

Wipe all demo data for this record

Deletes the msdyn_salespremiumdemo row plus every related lifetime / daily KPI / most-contacted / WKW row that points at this record. On opportunities, it also clears won-opportunity KPI rows loaded for Similar won comparisons, because those rows can still feed the visual after this record is wiped. A blank replacement row is then created so the native Sales Premium visual does not fall back to stale sample data.
